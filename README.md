# Meli Challenge

Built with:
```
NextJS
ReactJS
NodeJS
Redux
SASS
Jest
React-testing-library
```

## Frontend setup

```
cd frontend
npm i
npm run dev
open http://localhost:3000/
```

## Backend Setup

```
cd backend
npm i
npm start
```

## Testing

```
cd frontend
npm test
```
